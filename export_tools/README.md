# 导出工具集合

本文件夹包含用于从Unreal Engine导出各种格式数据的Python脚本。

## 📁 文件列表

- **`export_to_image.py`** - 渲染目标导出为图像文件
- **`sequence.py`** - Level Sequence相关工具

## 🎯 主要功能

### 图像导出工具 (`export_to_image.py`)
支持将渲染目标和纹理导出为各种图像格式：

#### ✨ 核心特性
- **多格式支持**: PNG、HDR、EXR等格式
- **高质量输出**: 支持浮点精度和线性色彩空间
- **自定义路径**: 灵活的输出路径配置
- **批量处理**: 支持多个纹理批量导出

#### 🔧 支持格式
- **PNG**: 标准图像格式，适用于一般用途
- **HDR**: 高动态范围格式，适用于光照分析
- **EXR**: 专业级格式，支持多通道和浮点精度

### 序列工具 (`sequence.py`)
Level Sequence相关的导出和处理工具。

## 🚀 使用方法

### 图像导出
```python
# 导出渲染目标为图像
exec(open("export_tools/export_to_image.py").read())
```

### 自定义导出
编辑脚本中的路径配置：
```python
# 定义导出路径
export_path = r"C:\Your\Custom\Path\output.hdr"

# 定义纹理路径
texture_path = "/Game/Your/Texture/Asset"
```

## ⚙️ 配置选项

### 导出参数
- **文件路径**: 自定义输出目录和文件名
- **纹理资产**: 指定要导出的纹理或渲染目标
- **格式选择**: 根据需求选择合适的输出格式
- **质量设置**: 调整压缩和质量参数

### 批量导出
```python
# 示例：批量导出多个纹理
textures = [
    "/Game/Texture1",
    "/Game/Texture2",
    "/Game/Texture3"
]

for texture_path in textures:
    # 导出逻辑
    pass
```

## 📋 应用场景

### 🎬 影视制作
- **渲染帧导出**: 从渲染目标导出高质量帧
- **光照贴图**: 导出光照分析结果
- **材质预览**: 生成材质库预览图

- **效果图导出**: 高分辨率静态图像

- **材质展示**: 材质库标准化

### 🎮 游戏开发
- **纹理资源**: 批量处理纹理资源
- **UI资源**: 导出UI元素

## 🔧 技术要求

### 系统要求
- **Unreal Engine**: 4.27+ / 5.0+
- **内存空间**: 大格式导出需要充足内存
- **磁盘空间**: 高分辨率输出占用较大空间

### 性能优化
- **分批处理**: 避免同时导出大量大尺寸图像
- **格式选择**: 根据用途选择合适的格式
- **分辨率控制**: 合理设置输出分辨率

## ⚠️ 注意事项

### 🔴 重要提醒
- **磁盘空间**: HDR和EXR格式文件较大
- **内存管理**: 大尺寸导出可能消耗大量内存
- **路径权限**: 确保输出目录有写入权限

### ✅ 最佳实践
1. **路径规划**: 提前规划好输出目录结构
2. **格式选择**: 根据最终用途选择合适格式
3. **批量测试**: 先小批量测试再大批量处理
4. **文件命名**: 使用清晰的文件命名规范

## 🐛 故障排除

### 常见问题
1. **权限错误**: 检查输出目录的写入权限
2. **内存不足**: 降低分辨率或分批处理
3. **路径错误**: 确认纹理资产路径正确
4. **格式不支持**: 检查目标格式是否支持

### 调试方法
- 检查UE编辑器输出窗口的错误信息
- 验证纹理资产路径是否正确
- 确认输出目录存在且可写
- 测试小分辨率导出是否正常
